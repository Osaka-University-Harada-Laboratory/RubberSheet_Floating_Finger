## パーツ

### 近接覚センサ：TK-01
製品URL：https://www.thinker-robotics.co.jp/

### センサカバー
該当CADファイル：\
作成方法：研究室では以下の2種類の3Dプリンターで作成．いずれの場合でも運用可能であることを確認．
  1. **光造形プリンター (AGILISTA-3200)**
     - モデル材：AR-M2
     - サポート材：AR-S1
  1. **FDM式3Dプリンター (Zortrax M200 Plus)**
     - フィラメント：Z-URTLAT (ABS系樹脂)
